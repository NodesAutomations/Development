### Create New Cuix File from Scrach
- Type `cui` command to load Custom User Interface
- Go to Transfer tab and Create New Customization file, Save it on your prefered location
- Now Go back to Customize tab, Find Partial Customization file in tree view, right click on it and choose load partial customization file
- Load your Cuix File
