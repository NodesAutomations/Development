## Find Keyboard Shortcuts

### General
- <kbd>?</kbd> : Open Up Short Cut Menu

## Most Usefull to me

### Search Code/Text inside Repo
- Go to repo you want to search for
- Press `/` to activate search bar
- Type `searchText in:file` in search bar 

### Fuzzy Serch to find any file in repo
- Open repo Main Page
- Press <kbd>T</kbd> on keyboard
- Start typing to Search

### Navigation
- <kbd>g</kbd><kbd>c</kbd> : Go to code
- <kbd>g</kbd><kbd>i</kbd> : Go to Issues

### Issues
- <kbd>c</kbd> : Create New Issue
