## Tips and Tricks

- User `start-transcript` Command in Start to Keep Log of All Activity in that Session
- Everything in **get/set/start/stop/update-Command** Format
- <mark style="background-color:#FFFF99;color:black">use get-help commandName for help related to That Command</mark>
- use `get-help commandName -examples` to see Examples related to that command
- use `get-help commandName -online` to see examples in Browser

## Command List

- **open file from Powershell : `Invoke-Item filepathWithExt`**
- Change Directory : `cd directoryName`
- Clear Screen : `cls`
- Check Version of installed Software : `SoftwareName --version`
- To get list of Command used in that session use `get-history`
