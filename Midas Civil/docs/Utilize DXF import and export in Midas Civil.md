### Summary

- Generate 3D model in Cad with Separate Layer for each Structure Group
- Create New Project in Midas and Set Units
- File—>Import—>Autocad Dxf File
- Now Load You Dxf File so you can select specific layers from it
- Midas Civil Has Multiple Ways to Import Autocad File
    - You Can import All Layer at Once
    - You Can import All Selected layers at once
    - You can import Layers you want to import one by one
    - You can import layers from drawing 1 and than Load another Drawing and Import Layers from Them
