# **Midas Civil Files Types**

### ***.mcb (binary)**

- Basic Data file for Midas Civil

### ***.mct (text)**

- Midas Text file for MCT Command Shell
- we can export mct file for model using File>Export>mct file

### ***.ca1 (binary)**

- Data file obtained from a static/dynamic analysis process

### ***.ca2 (binary)**

- Analysis results generated for each time step from a time history analysis and a heat of hydration analysis

### ***.ca3 (binary)**

- File for all the data obtained from a moving load analysis, influence line/influence surface and support ettlement analyses

### ***.ca4 (binary)**

- File for all the analysis data generated in the process of a geometric nonlinear analysis

### ***.ca5 (binary)**

- File for all the analysis data generated in the process of a pushover analysis

### ***.ca6 (binary)**

- File for all the analysis data generated in the process of construction stage analysis

### ***.anl (Text)**

- File containing structural analysis results (reactions, displacements, element forces, stresses, etc.) which has been arranged by the user’s preference. This file is useful for verifying analysis results and preparing calculation sheets.

### ***.out (Text)**

All kinds of messages or related data produced during a structural analysis process

### ***.cd8 (binary)**

- File for all the design data generated in the process of PSC design

### ***.color (binary)**

- Color data file of midas Civil

### ***.emf (binary)**

- Graphic data file of the model window in the EMF (Enhanced Meta File) format

### ***.bmp (binary)**

- Graphic data file of the model window in the BMP (Bitmap) format

### ***.jpg (binary)**

- Graphic data file of the model window in the JPEG format

### ***.mgf (binary)**
```
 - Graphic data file produced by Graphic Editor of midas Civil
```
