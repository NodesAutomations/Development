### Search withing specific profitl
```
from:profileName "searchString"
```
### Search but exclude some topics
```
from:profileName "searchString" -excludeItem1 -exludeItem2 -exludeItem3
```

### Search with min favorites
```
from:profileName min_faves:25
```

### Search with min retweets
```
from:profileName min_retweets:10
```
